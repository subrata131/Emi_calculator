# 💰 EMI Calculator

A simple and efficient **EMI (Equated Monthly Installment) Calculator** built using **Python**. This project calculates the monthly EMI, total payment, and total interest based on the loan amount, annual interest rate, and loan tenure.

---

## 📌 Features

* 💵 Calculate Monthly EMI
* 📊 Calculate Total Payment
* 💰 Calculate Total Interest
* 🖥️ Easy-to-use Command Line Interface (CLI)
* ⚡ Fast and accurate calculations
* 📝 Clean and beginner-friendly Python code

---

## 🛠️ Technologies Used

* Python 3
* Mathematical Formula
* Command Line Interface (CLI)

---

## 📐 EMI Formula

[
EMI = \frac{P \times R \times (1+R)^N}{(1+R)^N - 1}
]

Where:

* **P** = Loan Amount
* **R** = Monthly Interest Rate (Annual Rate ÷ 12 ÷ 100)
* **N** = Loan Tenure in Months
